## Inventory & Order Management System

Backend-first REST API built using Spring Boot and PostgreSQL.

## Features
- Product management
- Inventory tracking
- Transactional order creation
- Stock validation & rollback
- Product image upload (filesystem)
- RESTful API design

## Tech Stack
- Java, Spring Boot
- Spring Data JPA, Hibernate
- PostgreSQL
- Postman

## Run Locally
- Configure PostgreSQL in application.properties
- Run Spring Boot application
